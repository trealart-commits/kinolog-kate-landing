# KINolog Kate Landing

Одностраничный лендинг для кинолога Екатерины Соловьёвой.

## Запуск

```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000`.

## Структура

- `public/index.html` - основная страница
- `public/styles.css` - стили
- `public/script.js` - клиентский JS
- `public/assets/kinolog/` - изображения и сертификаты

## Публикация

В проект уже добавлен `render.yaml`, так что его можно задеплоить как отдельный сервис без связи с другими сайтами.
